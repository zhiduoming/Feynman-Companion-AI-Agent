## 一、调研背景与核心目标

### 1\.1 调研背景

1. **考研市场仍处高位但需求结构升级，从盲目报考转向高效备考**

    - 2025 年全国硕士研究生报考人数达 388万，2024年报考人数为438万，2025年较2024年下降50万人，降幅11\.4%，报录比仍处于高位，热门专业竞争尤为激烈（教育部，2024）

    - 2024年二战及以上往届考生占比达51\.82%，首次超过应届生，复读备考群体对高效提分、精准复习的需求显著高于应届生（中国教育在线，2025）

2. **传统考研学习模式陷入效率瓶颈**

    - 绝大多数考研学生存在"看书懂了，做题不会"的问题，超过半数学生认为自己"知识点背得熟，但主观题答不到点上"

    - 输入式学习在传统考研模式中占比过高，输出训练严重不足，导致"理解幻觉"成为考研学习最大的隐形杀手

3. **AI 技术为个性化学习提供新可能**

    - AI\+教育市场近年来保持快速增长，大语言模型在知识讲解、答疑、批改等场景的应用成熟度显著提升

    - AI Agent 技术的发展，使得"智能陪练"和"个性化反馈"成为可能，为解决传统费曼学习法的应用局限提供了技术基础

4. **费曼学习法的认知度与应用缺口**

    - 多数考研学生听说过费曼学习法，但能坚持使用的比例极低，主要障碍是"**缺少陪练对象"**和**"不知道自己讲得对不对"**

### 1\.2 核心调研目标

1. 系统拆解考研全流程学习行为，精准定位不同阶段、不同类型用户的核心痛点

2. 明确费曼学习法在考研各学科、各阶段的适配性，量化其解决核心痛点的价值

3. 基于 AI Agent 核心能力，设计可落地的 "AI \+ 费曼" 考研学习产品功能体系

4. 划分典型用户群体，为后续产品差异化设计提供依据

### 2\.2 核心关注优先级排序（按重要性）

1. **一级关注**

    - 知识点理解深度（尤其是数学定理、专业课抽象概念）

    - 主观题得分能力（论述题、简答题、案例分析题）

    - 复习精准度（避免无效学习，聚焦高频考点）

    - 分数提升效率（单位时间内的提分效果）

2. **二级关注**

    - 错题复盘效率

    - 背诵记忆效果

    - 学习计划合理性

    - 时间管理能力

3. **三级关注**

    - 学习动力与自律性

    - 信息获取便捷性

    - 同伴交流与支持

    - 学习环境舒适度

### 2\.3 典型用户分群（按核心学习痛点划分）

1. **概念理解困难型**特征：跨考/基础薄弱，对抽象原理和专业术语似懂非懂，只能死记硬背

2. 典型行为：反复看网课但仍不理解，做题时无法应用知识点

3. 核心需求：用通俗语言拆解复杂概念，建立知识体系

4. **输出能力薄弱型**特征：知识点背得熟，但主观题不会组织语言，抓不住得分点

5. 典型行为：背诵时能流利复述，答题时逻辑混乱，写了很多但得分低

6. 核心需求：训练语言组织能力和逻辑表达能力，掌握答题技巧

7. **知识碎片化型**特征：能记住零散知识点，但无法建立前后逻辑联系，形成知识孤岛

8. 典型行为：做选择题正确率高，但做综合题时无法融会贯通

9. 核心需求：重构知识体系，建立知识点之间的逻辑关联

10. **盲目刷题型**特征：大量刷题但不总结，同类题目反复出错

11. 典型行为：追求刷题数量，错题本只抄题不分析，不理解错误原因

12. 核心需求：精准定位知识漏洞，针对性强化训练

13. **自律性差型**特征：学习计划难以执行，容易分心，需要外部监督

14. 典型行为：每天学习时间长但效率低，经常拖延

15. 核心需求：学习监督、进度跟踪、及时反馈

## 三、考研全流程学习任务与行为拆解

### 3\.1 前期准备阶段（择校择专业 \- 3 月前）

- **核心任务**：信息收集、目标院校确定、基础水平摸底、学习计划制定

- **典型行为**：

    - 每天花费 1\-2 小时逛考研论坛、知乎、小红书，收集院校信息

    - 咨询学长学姐，了解报录比、考试难度、就业情况

    - 做 1\-2 套历年真题，评估自己的基础水平

    - 购买教材和辅导资料，制定全年学习计划

- **主要痛点**：信息不对称，容易被虚假信息误导；基础评估不准确，学习计划不合理

### 3\.2 基础夯实阶段（3\-6 月）

- **核心任务**：教材通读、核心概念理解、初步知识框架搭建

- **典型行为**：

    - 每天看 2\-3 小时网课，边看边划教材重点

    - 做章节笔记，整理知识点框架

    - 做课后习题和基础练习题

    - 每天背诵英语单词和政治基础知识点

- **主要痛点**：

    - 看书听课觉得懂了，合上书无法复述核心内容（理解幻觉）

    - 对专业术语和抽象原理理解不透彻，只能死记硬背

    - 知识碎片化，无法形成完整的知识体系

### 3\.3 强化提升阶段（7\-10 月）

- **核心任务**：知识点深化、专题训练、大量刷题、错题复盘

- **典型行为**：

    - 刷强化班网课，学习解题技巧和方法

    - 大量做真题和模拟题，每天刷题时间 3\-4 小时

    - 整理错题本，分析错误原因

    - 开始背诵专业课和政治核心考点

- **主要痛点**：

    - 错题复盘效率低，不知道自己为什么错

    - 知识点听懂了，但不会应用到具体题目中

    - 背诵内容多，容易遗忘，记忆不牢固

### 3\.4 冲刺模考阶段（11\-12 月）

- **核心任务**：全真模考、主观题专项训练、押题背诵、心态调整

- **典型行为**：

    - 每周进行 1\-2 次全真模考，严格按照考试时间答题

    - 专项练习论述题、简答题、案例分析题

    - 背诵押题卷和核心考点，每天背诵时间 4\-5 小时

    - 调整作息，保持良好心态

- **主要痛点**：

    - 主观题答题逻辑混乱，得分点不全

    - 答题速度慢，考试时做不完题目

    - 复习无重点，不知道哪些知识点需要重点掌握

    - 心态焦虑，容易放弃

### 3\.5 考后阶段（12 月 \- 次年 3 月）

- **核心任务**：估分、复试准备、调剂信息收集

- **典型行为**：

    - 对照参考答案估分，判断是否能进入复试

    - 准备复试专业课和英语口语

    - 关注调剂信息，联系调剂院校

- **主要痛点**：复试信息不对称，不知道如何准备；调剂信息获取不及时

## 四、考研用户核心痛点调研（按费曼适配性分类）

### 4\.1 输入阶段痛点（费曼学习法核心解决场景）

1. **理解幻觉**：多数考研学生表示"看书时觉得逻辑通顺，合上书就讲不清楚"。这是输入式学习最典型的问题，用户通过阅读和听课获得的只是"熟悉感"，而非真正的理解。

2. **概念模糊**：大量学生表示"对专业术语和抽象原理只能死记硬背，无法解释本质"。尤其是数学中的定理推导、经济学中的模型、法学中的法律条文，很多学生只能记住结论，不知道为什么是这样。

3. **知识碎片化**：不少学生表示"知识点孤立存在，无法建立前后逻辑联系"。传统的线性学习方式容易导致用户只关注单个知识点，而忽略了知识点之间的关联，形成知识孤岛。

4. **信息过载**：很多学生表示"网课和资料太多，抓不住重点"。考研市场上的辅导资料层出不穷，用户不知道哪些是核心内容，哪些是次要内容，导致学习效率低下。

### 4\.2 内化阶段痛点

1. **缺乏有效反馈**：绝大多数学生表示"自己讲给自己听，无法发现理解漏洞和逻辑跳跃"。传统的学习方式中，用户很难获得及时、准确的反馈，不知道自己的理解哪里有问题。

2. **反馈不专业**：多数学生表示"同伴水平有限，无法精准指出知识错误和逻辑问题"。即使找到同学一起学习，也很难获得专业的反馈，很多时候只是互相安慰，无法真正解决问题。

3. **迁移能力差**：大量学生表示"知识点听懂了，但不会应用到具体题目中"。这是因为用户没有真正理解知识点的本质，只是记住了表面内容，无法将其迁移到新的场景中。

4. **复习无重点**：不少学生表示"不知道自己哪些知识点掌握不牢，盲目全面复习"。由于无法精准定位知识漏洞，用户只能进行全面复习，导致时间浪费在已经掌握的知识点上。

### 4\.3 输出阶段痛点

1. **主观题答题困难**：多数学生表示"只会背诵知识点，不会组织语言，答题逻辑混乱"。主观题在考研中占比很高（政治50%、专业课60%\-100%），但很多学生缺乏系统的输出训练，导致主观题得分低。

2. **表达不精准**：大量学生表示"抓不住得分点，写了很多但得不到高分"。考研主观题有明确的评分标准，用户需要用精准的语言表达核心观点，但很多学生表达啰嗦，重点不突出。

3. **记忆不牢固**：被动背诵容易遗忘，无法精准复述核心内容。主动输出的记忆留存率显著高于被动记忆，这也是费曼学习法的核心优势之一。

4. **时间不够用**：不少学生表示"主观题答题速度慢，考试时做不完题目"。这是因为用户平时缺乏输出训练，语言组织能力和逻辑表达能力不足，导致答题速度慢。

### 4\.4 传统费曼学习法的应用局限

1. **缺少陪练对象**：绝大多数学生表示"很难找到愿意耐心听讲解并持续追问的人"。费曼学习法的核心是"教给别人"，但现实中很少有人愿意花费大量时间听用户讲解知识点。

2. **时间成本高**：多数学生表示"每次讲解需要占用他人大量时间，无法随时进行"。考研学生的时间非常宝贵，很难抽出大量时间和同学进行费曼对练。

3. **无法量化评估**：大量学生表示"不知道自己的讲解到底哪里有问题，改进方向不明确"。传统的费曼学习法没有明确的评估标准，用户无法准确判断自己的理解程度。

4. **难以形成闭环**：不少学生表示"讲解中暴露的问题无法自动纳入复习计划"。很多用户在讲解中发现了自己的知识漏洞，但没有及时记录和复习，导致问题再次出现。

## 五、费曼学习法在考研场景的适配性分析

### 5\.1 费曼学习法核心原理

费曼学习法由诺贝尔物理学奖获得者理查德・费曼提出，核心逻辑是**输出倒逼输入**，通过 "教给别人" 的方式暴露理解漏洞，从而深化对知识的理解。其经典步骤为：

1. **选择概念**：选择一个想要学习的知识点或概念

2. **教给小白**：假装把这个概念教给一个 8 岁的孩子，用最简单的语言解释清楚

3. **发现漏洞**：在讲解过程中，遇到卡壳、含糊不清或无法解释的地方，就是自己的知识漏洞

4. **回顾简化**：回到原始材料，重新学习漏洞部分，然后用更简单、更通俗的语言重新讲解

### 5\.2 费曼学习法解决考研痛点的核心价值

|**考研痛点**|**费曼学习法的解决方式**|**核心价值**|
|---|---|---|
|理解幻觉|要求用自己的话讲解，快速暴露"似懂非懂"|显著提升理解深度，打破"熟悉感"错觉|
|概念模糊|用通俗语言解释专业术语，拆解抽象原理|大幅提高概念掌握准确率，避免死记硬背|
|知识碎片化|通过讲解重构知识体系，建立逻辑联系|完善知识体系完整性，实现融会贯通|
|输出能力弱|训练语言组织能力和逻辑表达能力|有效提升主观题得分能力，掌握答题逻辑|
|记忆不牢|主动输出比被动记忆的留存率高得多|大幅提高知识点留存率，减少遗忘|
|复习盲目|通过追问精准定位知识盲区，明确复习重点|显著提升复习效率，避免无效学习|

### 5\.3 费曼学习法的考研适用场景边界

1. **最适用场景**

    - 概念理解：数学定理、专业课抽象概念、政治原理

    - 主观题训练：论述题、简答题、案例分析题

    - 错题复盘：分析错误原因，理解解题思路

    - 知识体系构建：梳理章节逻辑，建立知识框架

2. **较适用场景**

    - 背诵记忆：通过讲解加深对背诵内容的理解

    - 真题解析：讲解真题的解题思路和得分点

    - 专题总结：归纳总结某一专题的知识点和解题方法

3. **不适用场景**

    - 纯计算类题目训练：如数学中的计算题、英语中的语法题

    - 纯记忆类知识点：如英语单词、历史年代、政治时事

    - 技能类训练：如英语口语、听力

## 六、AI Agent 赋能费曼学习法的考研应用方向

### 6\.1 AI Agent 核心能力支撑

基于菜鸟教程对 AI Agent 的定义，AI Agent = LLM（大脑）\+ Planning（规划）\+ Tool use（执行）\+ Memory（记忆），其在费曼学习法中的核心支撑能力为：

1. **LLM 大脑**：自然语言理解、知识推理、生成追问和反馈，能够理解用户的讲解内容，识别其中的问题

2. **规划能力**：根据用户的学习水平和目标，制定个性化的费曼训练计划，合理安排训练内容和频率

3. **工具使用**：调用考研题库、知识点库、错题本、真题解析等工具，为用户提供全面的学习支持

4. **记忆系统**：记录用户的讲解过程、知识漏洞、复习进度和成长轨迹，实现个性化学习

### 6\.2 核心赋能功能

1. **智能费曼陪练**

    - 扮演 "零基础小白" 角色，针对用户讲解中的专业术语、逻辑跳跃、含糊不清处连续追问

    - 支持语音、文字、手写多模态输入讲解，满足不同学习场景的需求

    - 针对不同学科定制追问逻辑：数学重推导过程、政治重逻辑关系、专业课重概念本质

    - 示例：用户讲解 "边际效用递减规律" 时，AI 会追问："什么是边际效用？能不能举一个生活中的例子？为什么边际效用会递减？有没有例外情况？"

2. **自动费曼诊断报告**

    - 评估维度：理解深度、表达完整性、逻辑连贯性、结构化能力

    - 具体指出：哪些概念没讲透、哪些地方逻辑跳跃、哪些前提被省略、哪些例子不恰当

    - 对比标准讲解，给出优化建议和改进方向

    - 参考木铎笔记费曼模式，生成 "小白刁难点 \- 用户回应漏洞 \- 深度剖析" 三维诊断报告

3. **个性化复习闭环**

    - 将讲解中暴露的知识漏洞自动加入错题本和复习计划

## 七、核心应用场景详细描述

### 7\.1 基础阶段：概念理解费曼训练场景

- **用户画像**：跨考学生、基础薄弱学生

- **用户行为**：看完一章教材 / 一节网课后，尝试讲解核心概念

- **核心痛点**：概念模糊，无法用通俗语言解释，不知道自己哪里没懂

- **AI 解决方案**：

    1. 用户选择要讲解的概念（如 "剩余价值"）

    2. AI 引导用户："请用你自己的话，给一个完全没学过经济学的人讲解一下什么是剩余价值"

    3. 用户开始讲解（语音 / 文字）："剩余价值就是工人创造的被资本家无偿占有的价值"

    4. AI 连续追问：

        - "什么是价值？工人创造的价值都包括哪些部分？"

        - "为什么资本家能无偿占有工人创造的价值？"

        - "能不能举一个具体的例子说明剩余价值是怎么产生的？"

    5. 用户在讲解过程中遇到卡壳，AI 提示用户回顾教材相关内容

    6. 讲解结束后，AI 生成概念理解诊断报告，指出用户的知识漏洞

    7. AI 推送相关知识点的补充讲解和练习题，帮助用户巩固理解

### 7\.2 强化阶段：错题费曼复盘场景

- **用户画像**：盲目刷题型学生、知识碎片化型学生

- **用户行为**：做完一道错题后，尝试讲解解题思路

- **核心痛点**：不知道自己为什么错，知识点应用不熟练

- **AI 解决方案**：

    1. 用户上传错题（拍照 / 文字输入）

    2. AI 引导用户："请讲解一下这道题的解题思路，以及你当时为什么会做错"

    3. 用户开始讲解："这道题我选了 A，正确答案是 B，我当时觉得 A 是对的，因为\.\.\."

    4. AI 连续追问：

        - "你为什么认为 A 选项是正确的？它的错误在哪里？"

        - "B 选项为什么是正确的？它涉及到哪个知识点？"

        - "如果把题目中的条件改一下，答案会是什么？"

    5. AI 对比标准解题过程，指出用户的逻辑漏洞和知识漏洞

    6. 生成错题复盘报告，关联相关知识点

    7. 推送同类题目进行强化训练，帮助用户掌握解题方法

### 7\.3 冲刺阶段：主观题费曼输出场景

- **用户画像**：输出能力薄弱型学生

- **用户行为**：练习论述题、简答题，尝试口头或书面作答

- **核心痛点**：答题逻辑混乱，得分点不全，表达不清晰

- **AI 解决方案**：

    1. 用户选择要练习的主观题（如 "论述中国特色社会主义进入新时代的历史意义"）

    2. AI 引导用户："请用费曼的方式讲解一下这道题的答题思路，你会从哪几个方面来回答？"

    3. 用户开始讲解答题思路："我会从三个方面来回答，第一是\.\.\. 第二是\.\.\. 第三是\.\.\."

    4. AI 连续追问：

        - "你为什么选择这三个方面？它们之间的逻辑关系是什么？"

        - "每个方面的核心得分点是什么？你会用哪些论据来支撑？"

        - "有没有遗漏的重要得分点？"

    5. 用户完成答题后，AI 批改答案，评估得分点覆盖情况和逻辑清晰度

    6. 生成答题优化建议和高分范文对比

    7. 提供同类题目的答题模板和技巧，帮助用户提升答题能力

## 九、调研结论与产品建议

### 9\.1 核心调研结论

1. **考研用户对 "高效学习" 和 "精准提分" 的需求强烈**，传统输入式学习模式的效率瓶颈日益凸显，输出训练严重不足是导致学习效果差的主要原因

2. **费曼学习法是解决考研核心痛点的有效方法**，但传统应用存在缺少陪练对象、时间成本高、无法量化评估等局限

3. **AI Agent 技术为费曼学习法的普及提供了可能**，通过智能陪练、自动诊断、个性化复习等功能，可以有效解决传统费曼学习法的应用局限

4. **考研市场缺乏专门的 "AI \+ 费曼" 学习产品**，现有产品的费曼相关功能不够完善，无法满足考研学生的需求
